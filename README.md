# EventCore — Sistema de Gestión de Eventos

## Descripción

Sistema web interno para la planificación, organización y seguimiento de eventos. Permite gestionar usuarios con roles diferenciados, crear y administrar eventos con control de sedes, cupos, categorías y estados, todo desde un panel de control centralizado con interfaz premium dark-mode.

## Problema que Resuelve

La coordinación de eventos involucra múltiples tareas dispersas: control de accesos, asignación de responsabilidades, seguimiento de inscripciones, gestión de sedes y monitoreo del estado de cada evento. Sin una herramienta centralizada, esta información se pierde entre hojas de cálculo, correos y notas sueltas, generando duplicidad de datos, falta de visibilidad y errores operativos.

EventCore centraliza toda la logística en una sola plataforma, eliminando la desorganización y brindando visibilidad en tiempo real sobre la agenda de eventos.

## Usuarios Objetivos

| Rol | Descripción |
|-----|-------------|
| **Administrador** | Acceso total al sistema. Gestiona usuarios, asigna roles y supervisa todos los módulos. |
| **Organizador** | Crea y administra eventos propios, gestiona sedes y da seguimiento a la logística. |
